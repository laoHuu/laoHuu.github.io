发表于 2018-10-31 | 更新于 2019-03-13 | 分类于 Unity | | 阅读次数：

最近在编写unity的工具，碰见需要将工程中某个文件夹下的playable文件遍历出来让人选取，然后再进行操作，所以正好去了解了一下Unity中TreeView的用法，发现真的是很方便，所以做了个方便拓展的专门用于遍历目录文件的TreeView。

你们可以在工程中添加下面两个文件，然后在菜单栏中选中TreeView Examples/File Tree Window就可以查看效果。
